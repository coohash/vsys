# 🌐 在 Windows 系统新手配置指南
# ------------------------------------------------------------------------------
# 【第一步：下载与安装 Python 运行环境】
# 1. 访问 Python 官方网站下载页面： https://www.python.org/downloads/windows/
# 2. 找到 "Python 3.11.9" 或 3.11 系列版（不要下载最新版 3.13+，考虑第三方库底层 C 编译兼容性问题），下载 "Windows installer (64-bit)" 安装包。
# 3. 双击运行下载好的 `.exe` 文件。
# 4. 【🔥 核心关键点】：在安装窗口的最下方，务必【勾选】 "Add python.exe to PATH"（将Python加入环境变量）。
#    如果不勾选此项，后续在终端输入 python 或 pip 命令时会报错“不是内部或外部命令”。
# 5. 点击 "Install Now" 直到安装完成。
#
# 【第二步：打开 Windows 终端命令行】
# 1. 按下键盘上的快捷键： `Win + R` （会弹出一个“运行”小窗口）。
# 2. 在输入框内输入： `cmd` 
# 3. 点击“确定”或敲击回车，即可打开黑色的命令提示符（Terminal）窗口。
#
# 【第三步：验证安装是否成功】
# 1. 在黑色的命令行窗口中，输入以下命令并回车：
#    python --version
# 2. 如果正确显示类似 `Python 3.11.x` 的字样，说明环境配置成功！
#
# 【第四步：创建隔离的沙盒环境（防多项目代码污染）】
# 1. 在黑色的命令行窗口中，使用 `cd` 命令切换到您存放本代码仓库的文件夹目录下。例如：
#    cd C:\Users\YourName\Desktop\VSYS-Blockchain-Python-Code
# 2. 执行以下命令创建属于本项目的虚拟沙盒（venv）：
#    python -m venv venv
# 3. 激活这个虚拟环境：
#    - 如果是标准的 CMD 窗口，执行：  .\venv\Scripts\activate
#    - 如果是 PowerShell 窗口，执行： .\venv\Scripts\Activate.ps1
#    (激活成功后，您的命令行最前端会出现 `(venv)` 的括号标记。如果 PowerShell 报错“禁止运行脚本”，
#     请以管理员身份打开 PowerShell 并执行 `Set-ExecutionPolicy RemoteSigned` 放行)
#
# 【第五步：一键安装项目核心资产包】
# 1. 确保已经处于项目的根目录下，且命令行前端带有 `(venv)`。
# 2. 升级基础安装器：
#    python -m pip install --upgrade pip
# 3. 直接复制并运行下方指令，系统会自动剔除本指南的注释，精准安装最下方的依赖库：
#    pip install -r requirements.txt
#
